🚀 DESPLIEGUE EN 3 PASOS
═════════════════════════════════════════════════════════════════════════════

PASO 1: Agrega tus archivos
├─ background.mp3 → assets/audio/
├─ logo.png → assets/img/
└─ logo-large.png → assets/img/

PASO 2: Prueba localmente (opcional)
├─ Abre terminal en la carpeta
├─ Ejecuta: python -m http.server 8000
└─ Abre: http://localhost:8000

PASO 3: Sube a Vercel
├─ Ve a: https://vercel.com/new
├─ Opción A: Drag & Drop la carpeta (más fácil)
├─ Opción B: Conecta tu repo de GitHub
└─ ¡Tu web está en vivo! 🎉

🔧 PERSONALIZACIÓN DESPUÉS DE DESPLEGAR
═════════════════════════════════════════════════════════════════════════════

Editar sin perder datos:

PRECIOS
├─ Edita: js/data.js
├─ Busca: "price: 29000"
└─ Cambia por el nuevo precio

COLORES
├─ Edita: css/styles.css (primeras líneas)
├─ Variables:
│  ├─ --primary-dark: #0d1f2d
│  ├─ --accent-gold: #d4af37
│  └─ --text-light: #f5f5f5
└─ Guarda y Vercel redeploya automáticamente

CONTACTO (Instagram, teléfono, etc.)
├─ Edita: index.html
├─ Busca: "contacto" o "footer"
└─ Cambiar valores

AGREGAR PRODUCTOS
├─ Edita: js/data.js
├─ Agrega objeto nuevo siguiendo el formato
└─ Incluye: id, category, name, price, description, ingredients
